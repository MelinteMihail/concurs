Introducere in Flexbox

Introducere

Asa cum vei invata, exista multe modalitati de a muta elementele pe o pagina web. De-a lungul anilor, au fost dezvoltate metode noi, iar cele mai vechi au devenit invechite. Flexbox nu a fost intotdeauna disponibil in CSS—debutul sau a fost revolutionar. Afla mai multe despre istoria Flexbox.

Multe resurse il plaseaza spre sfarsitul curriculumului, deoarece este relativ nou ca tehnologie. Insa, in acest moment, a devenit metoda standard de pozitionare a elementelor pentru multi dezvoltatori. Flexbox va fi unul dintre cele mai utilizate instrumente din trusa ta de unelte, asa ca de ce sa nu-l inveti mai intai?

Inainte de a incepe

Layout-urile Flexbox pot deveni putin complicate. In lectia anterioara, ai invatat cum sa inspectezi si sa depanezi lucrurile folosind instrumentele pentru dezvoltatori ale browserului tau. Aceste instrumente vor fi esentiale in lectiile urmatoare. Daca ceva nu se comporta asa cum te astepti, inspectarea in instrumentele pentru dezvoltatori ar trebui sa fie primul pas de fiecare data.

Flexbox nu este neaparat mai dificil decat celelalte concepte pe care le-am acoperit pana acum, dar are cateva aspecte suplimentare. Va fi oarecum dificil sa folosesti orice lucru pe care il inveti in aceste prime lectii pana cand ajungi la final si le poti combina toate. Pe masura ce progresam, fa-ti un favor si experimenteaza cu toate exemplele de cod.

Aproape sigur va trebui sa revii si sa consulti aceste lectii (sau cateva dintre resursele pe care ti le oferim) atunci cand ajungi la temele de la sfarsitul sectiunii. Totusi, daca iti iei timp si experimentezi cu toate exemplele de cod pe care le oferim, vei sti mai bine unde sa cauti cand va fi nevoie.

Let's flex!

Flexbox este o metoda de organizare a elementelor in randuri sau coloane. Aceste elemente vor „flexa” (adica vor creste sau se vor micsora) in functie de anumite reguli pe care le poti defini. Pentru a incepe, sa aruncam o privire la o demonstratie.

Vom analiza exact ce se intampla aici in curand. Dar, pentru moment, sa decomentam cele doua declaratii CSS legate de flex din Codepen-ul de mai sus, eliminand /* si */ care le inconjoara, apoi sa verificam rezultatul.

Toate cele 3 div-uri ar trebui acum sa fie aranjate orizontal. Daca redimensionezi cadrul rezultatelor folosind butoanele „1x”, „.5x” si „.25x”, vei observa ca div-urile vor „flexa”. Ele vor umple spatiul disponibil si vor avea latime egala.

Daca adaugi un alt div in HTML, in interiorul .flex-container, acesta va aparea langa celelalte, iar toate elementele se vor adapta pentru a se potrivi in spatiul disponibil.

Containere si elemente flex

Asa cum ai observat, Flexbox nu este doar o singura proprietate CSS, ci un intreg set de instrumente pe care le poti folosi pentru a pozitiona elementele acolo unde ai nevoie. Unele dintre aceste proprietati apartin containerului flex, in timp ce altele sunt aplicate elementelor flex. Acesta este un concept important.

Un container flex este orice element care are display: flex aplicat. Un element flex este orice element care se afla direct in interiorul unui container flex.

Putin derutant, orice element poate fi atat un container flex, cat si un element flex. Cu alte cuvinte, poti aplica display: flex pe un element flex si apoi sa folosesti Flexbox pentru a aranja elementele sale copil.

Crearea si imbricarea mai multor containere si elemente flex este metoda principala prin care vom construi layout-uri complexe. Imaginea urmatoare a fost realizata folosind exclusiv Flexbox pentru a aranja, dimensiona si pozitiona diversele elemente. Flexbox este un instrument extrem de puternic.
