CSS Box Model

Introducere

Acum ca intelegi sintaxa de baza a HTML si CSS, este timpul sa devenim seriosi. Cele mai importante abilitati pe care trebuie sa le stapanesti in CSS sunt pozitionarea si layout-ul. Schimbarea fonturilor si culorilor este o abilitate esentiala, dar a putea plasa elementele exact unde doresti pe o pagina web este si mai crucial. La urma urmei, cate pagini web poti gasi unde fiecare element este pur si simplu stivuit unul peste altul?

Invatarea modului de pozitionare a elementelor pe o pagina web nu este dificila odata ce intelegi cateva concepte cheie. Din pacate, multi cursanti trec rapid prin HTML si CSS pentru a ajunge la JavaScript si sfarsesc prin a omite aceste concepte fundamentale. Acest lucru duce la frustrare si durere (si gif-uri amuzante), deoarece toate abilitatile de JavaScript din lume sunt inutile daca nu poti plasa elementele exact unde trebuie pe pagina. Asadar, cu acest lucru in minte, sa incepem!

Box model

Primul concept important pe care trebuie sa-l intelegi pentru a avea succes in CSS este modelul casetei. Nu este complicat, dar daca il ignori acum, vei avea multe frustrari mai tarziu.

Fiecare element de pe o pagina web este o caseta dreptunghiulara. Aceste casete pot contine alte casete in interiorul lor si pot fi asezate una langa alta. Poti obtine o idee generala despre cum functioneaza acest lucru aplicand un contur fiecarui element de pe pagina astfel: 

                
                    * {
                        outline: 2px solid red;
                    }

Poti folosi inspectorul browserului pentru a adauga CSS-ul de mai sus la aceasta pagina web, daca doresti, facand clic pe butonul „+” din coltul din dreapta sus al panoului „Styles” din fila „Elements”. Casete in casete!

OK, poate ca in imaginea de mai sus exista cateva cercuri... dar cand vine vorba de layout, ele se potrivesc impreuna ca niste casete dreptunghiulare, nu ca niste cercuri. In cele din urma, organizarea unei pagini web si pozitionarea tuturor elementelor inseamna deciderea modului in care vei incadra si stivui aceste casete.

Singura complicatie reala este ca exista multe modalitati de a manipula dimensiunea acestor casete si spatiul dintre ele, utilizand padding, border si margin. Articolele atribuite aprofundeaza acest concept, dar pe scurt:

  • padding creste spatiul dintre marginea unei casete si continutul acesteia.
  • border adauga spatiu (chiar daca este doar un pixel sau doi) intre margin si padding.
  • margin creste spatiul dintre marginile unei casete si marginile casetelor adiacente.
