Elemente si Etichete

Introducere

HTML (HyperText Markup Language) defineste structura si continutul paginilor web. Folosim elemente HTML pentru a crea toate paragrafele, titlurile, listele, imaginile si linkurile care formeaza o pagina web obisnuita. In aceasta lectie, vom explora cum functioneaza elementele HTML.

Elemente si etichete

Aproape toate elementele dintr-o pagina HTML sunt doar bucati de continut incadrate intre etichete HTML de deschidere si inchidere.

Etichetele de deschidere ii spun browserului ca aici incepe un element HTML. Ele sunt formate dintr-un cuvant-cheie incadrat intre paranteze unghiulare <>. De exemplu, o eticheta de deschidere pentru un paragraf arata asa: <p>.

Etichetele de inchidere ii spun browserului unde se termina un element. Ele sunt aproape identice cu cele de deschidere, cu diferenta ca au o bara oblica (slash) inaintea cuvantului-cheie. De exemplu, o eticheta de inchidere pentru un paragraf arata asa: </p>.

Un element complet de paragraf arata asa:

<p>sunt un paragraf</p>

Hai sa luam pas cu pas ce se intampla:

  • <p> este eticheta de deschidere.
  • sunt un paragraf reprezinta continutul incadrat intre eticheta de deschidere si cea de inchidere.
  • </p> este eticheta de inchidere.

Poti sa te gandesti la elemente ca la niste containere pentru continut. Etichetele de deschidere si inchidere spun browserului ce tip de continut se afla in interior. Browserul foloseste aceste informatii pentru a sti cum sa interpreteze si sa afiseze continutul.

HTML contine o lista foarte mare de etichete predefinite pe care le poti folosi pentru a crea tot felul de elemente diferite. Este important sa folosesti etichetele corecte pentru continutul tau. Alegerea potrivita a etichetelor poate avea un impact semnificativ asupra a doua aspecte ale site-ului tau: cum este clasat in motoarele de cautare si cat de accesibil este pentru utilizatorii care folosesc tehnologii asistive, cum ar fi cititoarele de ecran.

Elemente vid

Unele elemente HTML nu au o eticheta de inchidere. Aceste elemente au doar o singura eticheta, cum ar fi: <br> sau <img>. Ele sunt cunoscute drept elemente vid deoarece sunt goale, nu contin nimic in interior. Fara o eticheta de inchidere, ele nu pot incadra continut asa cum fac alte etichete.

Este posibil sa le mai vezi mentionate si ca etichete autoinchise. Dar acestea sunt tot elemente vid, doar ca au o bara oblica (/) la final, cum ar fi: <br /> sau <img />. Vei intalni destul de des aceste etichete autoinchise, din motive istorice. Browserele le vor afisa corect, insa cea mai recenta versiune a specificatiei HTML descurajeaza folosirea lor si le considera invalide.
