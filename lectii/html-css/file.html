Crearea unei file HTML

Introducere

Toate documentele HTML au aceeasi structura de baza (sau boilerplate), care trebuie sa fie prezenta inainte de a putea face ceva util. In aceasta lectie, vom explora diferitele parti ale acestui boilerplate si vom vedea cum se imbina toate intre ele.

Crearea unui fisier HTML

Pentru a demonstra un boilerplate HTML, mai intai avem nevoie de un fisier HTML cu care sa lucram.

Creeaza un folder nou pe calculatorul tau si numeste-l html-boilerplate. In interiorul acestui folder, creeaza un fisier nou si numeste-l index.html.

Vom folosi Visual Studio Code ca editor de text pentru a scrie si salva codul nostru HTML. Este un editor gratuit si foarte popular, folosit de multi dezvoltatori web.

Probabil esti deja familiar cu multe tipuri de fisiere, cum ar fi doc, pdf sau fisiere imagine.

Pentru a spune calculatorului ca vrem sa cream un fisier HTML, trebuie sa adaugam extensia .html la numele fisierului, asa cum am facut cand am creat fisierul index.html.

Merita mentionat ca am numit fisierul nostru HTML index. Ar trebui intotdeauna sa numim fisierul HTML care va contine pagina principala a site-ului nostru index.html. Asta pentru ca serverele web cauta in mod implicit o pagina index.html atunci cand utilizatorii acceseaza site-ul, iar lipsa acesteia poate cauza probleme mari.

DOCTYPE

Fiecare pagina HTML incepe cu o declaratie doctype. Scopul doctype-ului este sa ii spuna browserului ce versiune de HTML ar trebui sa foloseasca pentru a afisa documentul. Cea mai recenta versiune de HTML este HTML5, iar doctype-ul pentru aceasta versiune este: <!DOCTYPE html>.

Doctype-urile pentru versiunile mai vechi de HTML erau putin mai complicate. De exemplu, aceasta este declaratia doctype pentru HTML4:

<!DOCTYPE HTML PUBLIC "-//W3C//DTD HTML 4.01 Transitional//EN" "http://www.w3.org/TR/html4/loose.dtd">

Totusi, cel mai probabil nu vom dori niciodata sa folosim o versiune mai veche de HTML, asa ca vom folosi intotdeauna <!DOCTYPE html>.

Deschide fisierul index.html creat mai devreme in editorul tau de text si adauga <!DOCTYPE html> pe prima linie a fisierului.

Elementul HTML

Dupa ce declaram doctype-ul, trebuie sa adaugam un element <html>. Acesta este cunoscut ca elementul radacina al documentului, ceea ce inseamna ca orice alt element din document va fi un descendent al acestuia.

Acest lucru devine mai important mai tarziu, atunci cand vom invata cum sa manipulam HTML-ul folosind JavaScript. Pentru moment, este suficient sa stii ca elementul <html> ar trebui sa fie inclus in orice document HTML.

Revenind la fisierul index.html, haide sa adaugam elementul <html> scriind etichetele de deschidere si inchidere corespunzatoare: 

                
                    <!DOCTYPE html>
                    <html lang="en">
                    </html>

Ai observat cuvantul lang aici? El reprezinta un atribut HTML care este asociat cu eticheta HTML respectiva, in acest caz, <html>. Atributele HTML ofera informatii suplimentare despre elementele HTML.

Ce este atributul lang?

Atributul lang specifica limba continutului text din acel element. Acest atribut este folosit in principal pentru a imbunatati accesibilitatea paginii web. El permite tehnologiilor asistive, cum ar fi cititoarele de ecran, sa se adapteze in functie de limba si sa redea pronuntia corecta.

Elementul head

Elementul <head> este locul in care punem informatii meta importante despre paginile noastre web si alte lucruri necesare pentru ca acestea sa se afiseze corect in browser. In interiorul elementului <head> nu ar trebui sa folosim niciun element care afiseaza continut pe pagina web.

In fisierul index.html, haide sa adaugam un element <head> care sa contina un element <meta> si un titlu (<title>). Elementul <head> se adauga in interiorul elementului <html> si trebuie sa fie intotdeauna primul element dupa eticheta de deschidere <html>: 

                
                    <!DOCTYPE html>

                    <html lang="en">
                    <head>
                        <meta charset="UTF-8">
                        <title>Prima mea pagina web</title>
                    </head>
                    </html>

Elementul meta

Ar trebui sa avem intotdeauna eticheta <meta> cu setarea charset pentru codificarea caracterelor paginii in interiorul elementului <head>: <meta charset="UTF-8">.

Setarea codificarii este foarte importanta deoarece asigura afisarea corecta in browser a simbolurilor speciale si a caracterelor din diferite limbi.

Elementul title

Un alt element pe care ar trebui sa-l includem intotdeauna in sectiunea <head>; a unui document HTML este elementul <title>:

<title>Prima mea pagina web</title>

Elementul <title>; este folosit pentru a oferi o denumire usor de inteles paginii web, care este afisata in tab-ul browserului. De exemplu, daca te uiti acum la titlul tab-ului curent din browser, probabil vei vedea ceva de genul „Crearea unei file HTML”; acesta este titlul definit de elementul <title> din fisierul HTML.

Daca nu am include un element <title>, titlul paginii ar fi setat implicit la numele fisierului. In cazul nostru, acesta ar fi „index.html”, ceea ce nu este foarte util pentru utilizatori si ar ingreuna gasirea paginii daca sunt multe tab-uri deschise.

Exista multe alte elemente care pot fi incluse in sectiunea <head> a unui document HTML. Totusi, pentru moment este important sa cunosti doar cele doua prezentate aici. Vom introduce si alte elemente ale <head>-ului pe masura ce avansam in curs.

Elementul body

Elementul final necesar pentru a completa scheletul HTML este elementul <body>. Aici va fi tot continutul care va fi afisat utilizatorilor - textul, imaginile, listele, link-urile si asa mai departe.

Pentru a completa scheletul, adauga un element <body> in fisierul index.html. Elementul <body> se afla de asemenea in interiorul elementului <html> si este intotdeauna sub elementul <head>, astfel: 

                
                    <!DOCTYPE html>
                    <html lang="en">
                    <head>
                        <meta charset="UTF-8">
                        <title>Prima mea pagina web</title>
                    </head>

                    <body>
                    </body>
                    </html>

Vizualizarea fisierelor HTML in browser

Scheletul HTML din fisierul index.html este complet in acest moment, dar cum il poti vizualiza in browser? Exista cateva optiuni diferite:

  1. Poti trage si plasa un fisier HTML din editorul tau de text direct in bara de adresa a browser-ului.
  2. Poti gasi fisierul HTML in sistemul tau de fisiere si apoi sa dai dublu click pe el. Astfel, fisierul se va deschide in browser-ul implicit al sistemului tau.
  3. Poti folosi terminalul pentru a deschide fisierul in browser:
    • Ubuntu: Navigheaza in directorul care contine fisierul si tasteaza google-chrome index.html
    • macOS: Navigheaza in directorul care contine fisierul si tasteaza open ./index.html
    • WSL: Navigheaza in directorul care contine fisierul si tasteaza explorer.exe index.html. Atentie, acest lucru va deschide fisierul in browser-ul implicit al sistemului.

Folosind una dintre metodele de mai sus, deschide fisierul index.html la care am lucrat. Vei observa ca ecranul este gol. Acest lucru se intampla pentru ca nu avem nimic in body care sa se afiseze.

Revenind la fisierul index.html, sa adaugam un titlu in interiorul body-ului si sa salvam fisierul: 

                
                    <!DOCTYPE html>
                    <html lang="en">
                    <head>
                        <meta charset="UTF-8">
                        <title>Prima mea pagina web</title>
                    </head>

                    <body>
                        <h1>Hello World!</h1>
                    </body>
                    </html>

Acum, daca dai refresh la pagina in browser, ar trebui sa vezi modificarile aplicate, iar titlul „Hello World!” va fi afisat.
