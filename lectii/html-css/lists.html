Liste

Introducere

Fie ca este vorba despre top 250 de filme de pe IMDB sau despre cei mai cautati de FBI, listele sunt peste tot pe web si, mai devreme sau mai tarziu, vei avea nevoie de ele in paginile tale web.

Din fericire, HTML iti pune la dispozitie cateva tipuri diferite de liste.

Liste neordonate

Daca vrei sa ai o lista de elemente in care ordinea nu conteaza, cum ar fi o lista de cumparaturi in care obiectele pot fi achizitionate in orice ordine, atunci poti folosi o lista neordonata.

Listele neordonate sunt create utilizand elementul <ul>, iar fiecare element din lista este creat folosind elementul <li>.

Liste ordonate

Daca, in schimb, vrei sa creezi o lista de elemente in care ordinea conteaza, cum ar fi instructiuni pas cu pas pentru o reteta sau top 10 al emisiunilor TV preferate, atunci poti folosi o lista ordonata.

Listele ordonate sunt create folosind elementul <ol>. Fiecare element individual din acestea este, din nou, creat folosind elementul <li>. Totusi, fiecare element dintr-o lista ordonata incepe cu un numar, in loc de un punct de marcaj:
