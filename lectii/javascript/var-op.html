Introducere

In sectiunile anterioare, ai invatat cum sa structurezi paginile web folosind HTML si cum sa le stilizezi cu CSS. Urmatorul pas este sa facem pagina web interactiva, iar exact pentru acest lucru este destinat JavaScript.

In aceasta sectiune, ne vom concentra pe fundamentele JavaScript si pe modul in care il poti folosi pentru a manipula toate interactiunile dintre pagina web si utilizator. Pregatit sa dai viata codului tau?

Cum sa rulezi cod JavaScript

Majoritatea codului JavaScript pe care il vom scrie in acest curs va fi executat prin intermediul browserului. Cel mai simplu mod de a incepe este sa creezi un fisier HTML care contine codul JavaScript. Foloseste snipet-ul VS Code ! + TAB pentru a genera scheletul HTML de baza intr-un fisier de pe computerul tau. Asigura-te ca includi tag-ul <script>. Iata un exemplu: 

                
                    <!DOCTYPE html>
                    <html lang="en">
                        <head>
                        <meta charset="UTF-8">
                        <meta name="viewport" content="width=device-width, initial-scale=1.0">
                        <title>Document</title>
                    </head>
                    <body>

                        <script>
                            console.log("Hello, World!")
                        </script>

                    </body>
                    </html>

Odata ce ai salvat si deschis fisierul HTML intr-un browser, poti accesa consola pentru a vedea output-ul comenzii console.log.

Pasii pentru a deschide consola browserului:

  1. Click dreapta pe pagina web goala.
  2. Selecteaza “Inspect” sau “Inspect Element” pentru a deschide Developer Tools.
  3. Gaseste si selecteaza tab-ul “Console” → unde vei vedea mesajul "Hello, World!".

console.log() este comanda utilizata pentru a afisa ceva in consola dezvoltatorului din browserul tau. Poti folosi aceasta comanda pentru a printa rezultatele oricaruia dintre articolele si exercitiile urmatoare direct in consola. Te incurajam sa codezi alaturi de toate exemplele din aceasta si din lectiile viitoare!

O alta modalitate de includere a JavaScript-ului intr-o pagina web este prin intermediul unui script extern. Acest lucru este foarte similar cu linkarea unor documente CSS externe in site-ul tau. Pentru a incarca un script extern, poti folosi tag-ul <script> in HTML astfel: 

                
                    <script src="javascript.js"></script>

Fisierele JavaScript au extensia .js, similar cu .css pentru stiluri. Fisierele JavaScript externe sunt folosite pentru scripturi mai complexe.

Am numit fisierul javascript.js, dar am fi putut alege orice nume, cum ar fi my-script.js sau chiar fara nume. Cel mai important este extensia .js.

Variabile

Acestea sunt blocurile fundamentale ale oricarui program. Poti considera variabilele drept „containere de stocare” pentru datele din codul tau.

Poti declara variabile folosind cuvântul cheie let. 

                
                    let name = "John";
                    let surname = "Doe";

                    console.log(name);
                    console.log(surname);

Ce va afisa console.log? Incearca!

Poti de asemenea sa reatribui variabile: 

                
                    let age = 11;
                    console.log(age); // afiseaza 11

                    age = 54;
                    
                    console.log(age);

Observa lipsa lui let pe linia 4 - nu avem nevoie de el deoarece variabila a fost deja declarata mai devreme si doar ii reatribuim valoarea aici!

Reatribuirea este tare si toate cele, dar ce facem daca nu vrem sa se intample? De exemplu, am putea avea o constanta pi care nu va trebui niciodata reatribuita. Putem realiza acest lucru folosind cuvantul cheie const. 

                
                    const pi = 3.14;
                    pi = 10;

                    console.log(pi);

Intuitia ta poate sa iti spuna ca 3.14 va fi afisat. Incearca!

Este aruncata o eroare. Nici macar nu ajunge la console.log! Te poti intreba de ce am vrea o eroare in codul nostru. Adevarul este ca erorile sunt incredibil de utile pentru a ne spune ce este in neregula cu codul nostru si exact unde este problema. Fara ele, codul nostru tot nu ar face ce ne dorim, dar ar fi un chin sa incercam sa gasim ce este gresit!

Asadar, exista doua moduri de a declara o variabila:

  • let, pe care o putem reatribui.
  • const, pe care nu o putem reatribui si va arunca o eroare daca incercam.

Exista si un al treilea mod, var, care era modul original prin care variabilele erau declarate in JavaScript. var este similar cu let prin faptul ca variabilele declarate astfel pot fi reatribuite, dar are si alte particularitati care au fost rezolvate cand limbajul a introdus let si const. In mare parte, nu mai este folosit. Totusi, este posibil sa dai peste cod care foloseste var la un moment dat, asa ca este util sa stii ca exista.

Numerele

Numerele sunt elementele de baza ale logicii de programare. De fapt, este greu sa ne gandim la o sarcina de programare utila care sa nu implice macar putina matematica de baza... asa ca este evident ca intelegerea modului in care functioneaza numerele este foarte importanta. Din fericire, este si destul de simplu.

Daca ai fost la scoala, probabil nu vei gasi conceptul prea dificil de inteles. De exemplu, expresia matematica (3 + 2) - 76 * (1 + 1) este de asemenea un cod valid in JavaScript. Daca introduci asta intr-o comanda console.log, aceasta va evalua expresia si va afisa numarul corect. Incearca!
